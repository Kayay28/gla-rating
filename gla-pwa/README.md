# GLA Teacher Performance Rating System
## GreAtMinds Leadership Academy

---

## Quick Start

### Step 1 — Open the app
- Double-click `index.html` to open in browser
- Or host it on GitHub Pages for online access

### Step 2 — Enable Word Export (2-page guaranteed)
1. Double-click **`start.bat`** (Windows) — this starts the Word server
2. Keep that window open
3. Now click "Save as Word" in the app — it will generate a perfect 2-page document

### Step 3 — Use the app
- Fill in teacher info and ratings
- Click "Generate Rating Sheet"
- Print to PDF or Save as Word

---

## Files
- `index.html` — Main app (open this in browser)
- `server.py` — Python server for Word generation
- `start.bat` — Easy start script (Windows)
- `manifest.json` — PWA manifest
- `sw.js` — Service worker (offline support)
- `icons/` — App icons

---

## Install on Phone
1. Open `index.html` URL in Chrome (Android) or Safari (iOS)
2. Android: tap 3 dots → Add to Home Screen
3. iOS: tap Share → Add to Home Screen

---

## Deploy Online (GitHub Pages)
1. Create a GitHub repo
2. Upload all files
3. Enable GitHub Pages in Settings
4. Share the URL with anyone

---

## Features
- Dashboard with stats and recent ratings
- Rate 63 items across 4 sections
- Auto-save drafts
- Rating history with search & filter
- Teachers directory
- Reports & analytics
- Export to PDF, Word, CSV
- QR code sharing
- Works offline (PWA)
